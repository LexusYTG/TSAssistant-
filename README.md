# TS Assistant

Un asistente de voz completo para Android desarrollado en Java, diseñado específicamente para hablantes de español. Controla tu dispositivo mediante comandos de voz sin depender de servicios en la nube.

## Características principales

- Reconocimiento y síntesis de voz en español
- Más de 50 comandos diferentes
- Funcionamiento offline para comandos básicos
- Interfaz minimalista con visualizador de audio animado
- Procesamiento local de datos (privacidad garantizada)
- Sistema de caché inteligente para apps, contactos y música

## Comandos disponibles

**Control del dispositivo:**
- Abrir aplicaciones: "abre WhatsApp"
- Llamadas: "llama a mamá"
- Mensajes: "envía mensaje a Juan que diga hola"
- Sistema: "sube volumen", "aumenta brillo"

**Productividad:**
- Alarmas: "pon alarma para las 7"
- Temporizadores: "pon temporizador para 5 minutos"
- Recordatorios: "recuérdame comprar leche"
- Cálculos: "calcula 15 * 3"

**Información:**
- Hora y fecha: "qué hora es"
- Batería: "nivel batería"
- Búsquedas: "busca recetas de pizza"

**Entretenimiento:**
- Música: "reproduce Bohemian Rhapsody"
- Chistes: "cuenta un chiste"
- Comandos especiales: "pon dinamita"

**Navegación:**
- Direcciones: "navega a casa"
- Traducción: "traduce hola a inglés"

## Requisitos del sistema

- Android 4.0 (API 14) o superior
- Micrófono funcionando
- Servicio de reconocimiento de voz

## Instalación

1. Descarga el APK desde la sección de releases
2. Habilita "Fuentes desconocidas" en tu dispositivo
3. Instala la aplicación
4. Concede los permisos solicitados

## Uso básico

1. Inicia la aplicación
2. Espera a que se active el reconocimiento de voz
3. Di un comando en español
4. El asistente responderá por voz y ejecutará la acción

## Permisos requeridos

- Micrófono: Para escuchar comandos
- Contactos: Para llamadas y mensajes
- Teléfono: Para realizar llamadas
- Almacenamiento: Para acceder a música
- Otros permisos opcionales según funcionalidades usadas

## Detalles técnicos

Desarrollado en Java para Android, utiliza:
- SpeechRecognizer de Android para reconocimiento de voz
- TextToSpeech para síntesis de voz
- ContentResolver para acceso a contactos y medios
- MediaStore para biblioteca de música
- Animaciones personalizadas para interfaz
